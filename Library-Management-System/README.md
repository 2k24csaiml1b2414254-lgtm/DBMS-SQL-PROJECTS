# Library Management System

## 📚 Project Description
This project is a simple Library Management System built using SQL.  
It manages books, authors, members, and issued books in a library.

## 📂 Project Structure

Library-Management-System
│
├── schema.sql
├── sample_data.sql
├── queries.sql
├── ER-Diagram.png
└── README.md

## 🗄️ Database Tables

1. Authors
2. Books
3. Members
4. Issue

## ⚙️ Features

- Store book information
- Manage library members
- Issue books to members
- Track return dates
- Perform SQL queries

## 🛠 Technologies Used

- SQL
- MySQL
- DBMS concepts (ER Diagram, Schema, Queries)
